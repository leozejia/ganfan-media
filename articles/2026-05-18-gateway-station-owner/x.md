# 重生之我做中转站站长：我会怎么赚你的糊涂钱

重生之我做中转站站长。

我要赚钱，最好的办法不是明着骗你。明着骗，风险太高。我只需要略施小计，让你看不懂。

现在主流中转站都支持看使用记录了，每次请求用了哪个模型、花了多少钱、输入输出各多少，点开都能查。这些是明面上的东西，没什么操作空间。

但暗面的空间其实大得很。

我不会告诉你，你用的不是 Claude。

页面上写着 Claude，你以为跑的就是 Claude。但我后台可以把你的请求悄咪咪转到一个便宜得多的模型上。回答看着差不多，能力差一截，小白感知不到。但我收的还是 Claude 的钱。

同样的手法，写着 Opus 4.7，实际跑的可能是 Sonnet 4.5。技术上包装一下，你看到的模型名、返回的格式都一样。除非你对不同模型的输出风格很熟，不然分不出来。

怎么防？同一个问题，分别在中转站和官方跑一遍，对比输出质量和速度。如果差距明显，大概率不是同一个模型。

我不会告诉你，"官方"两个字是我自己贴的。

写着"官方接口""直连""官方高速"，你以为我从 Anthropic 或者 OpenAI 那里买了正规 API。但实际上我用的可能是逆向接口，就是用技术手段绕过官方授权搞出来的。成本比正规 API 低很多，但稳定性差，随时可能被封。

你用着用着突然断了，我跟你说"官方在维护"。你也没办法验证。

我不会告诉你，价格是我自己编的。

这个最隐蔽，要细说。

你在使用记录里能看到"倍率"，比如 0.2 倍、0.5 倍，看着挺便宜。但倍率是乘在一个底价上的，这个底价大部分平台不会直接告诉你，就算告诉你了，你确定是真实的定价吗？毕竟网页就是个展示壳子罢了。

举个例子。某模型官方输入价格 3 块（每百万 token），我后台偷偷把底价设成 8 块，然后给你一个 0.5 倍率。你一看，0.5 倍，这家真良心。但 8 乘 0.5 等于 4 块，比官方还贵。如果老实按官方 3 块算，0.5 倍应该是 1.5 块。

差价就在这。你只看到倍率低，觉得划算。但你不知道倍率乘的是什么数。

跑出来账单贵了你来问，我就说官方就这个价。你也没法查，因为大部分人根本不知道官方原价是多少。

最后一种你根本不会想到。

有些中转站价格低到离谱，甚至免费。你有没有想过，钱从哪来？

你发给 AI 的每一条消息、每一段提示词、你让它处理的文件和代码，都是数据。有人拿来做训练，有人拿来做分析。你觉得自己占了便宜，实际上你就是商品。

稳定、低价、高速，我们说不可能三角。

一个平台三个全占了，你得想想第四样东西是什么。

以上就是我赚糊涂钱的几个手段。模型名上动手脚，贴个"官方"标签，底价上做文章，或者直接拿你的数据补差价。

但不是所有中转站都这样。靠谱的平台会把每次请求的模型、用量、花费全摊开给你看，倍率和底价写得清清楚楚。能这样做的，至少说明它不怕你算账。
