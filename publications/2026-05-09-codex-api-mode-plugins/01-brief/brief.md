# Brief

## Audience

People using Codex App with API login who want plugins such as Browser, Chrome, Computer Use, or HyperFrames to work.

## Promise

After reading the X thread, they should understand the core fix: API mode is not the blocker; the plugin must be installed through Codex's plugin setup flow, restarted, and stale old browser automation may need cleanup.

## X Core Point

Codex App API mode can use plugins. The common failure is installing the wrong thing or leaving stale browser automation behind.

## SorryCode Action

SorryCode should carry the full checklist: which plugin to choose, install flow, config checks, Chrome extension/native host checks, HyperFrames marketplace name, stale agent-browser cleanup, and permission notes.

## Image Decision

No image needed for the first version. This is a practical fix thread; text has higher value than a forced cover.
