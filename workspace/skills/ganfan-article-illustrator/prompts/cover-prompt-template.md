# Cover Prompt Template

适用于文章头图。默认用于一张可复用的宽幅主图。

## 结构

```text
主体：{subject}
构图：{composition}
动作/状态：{action_or_tension}
场景：{location}
风格：{style_keywords}
光影：{lighting}
细节质感：{detail_texture}
技术参数：wide cover composition, central safe zone, important subject kept away from edges
不要：{negative_keywords}
```

## 使用原则

- 主体必须服务文章总论点，不做字面复述
- 构图默认宽幅、中心安全区、边缘不放关键信息
- 若文章偏方法论或商业分析，优先使用抽象隐喻、关系张力、信息图感
- 负面提示至少包含：文字、水印、logo、品牌标识、艺术家签名、版权信息
