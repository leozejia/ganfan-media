# Jarvis 语音克隆技术报告

## 克隆原理 (Qwen3-TTS-0.6B-Base)

### 技术架构

```
参考音频
    ↓
┌─────────────────────────────────────┐
│  1. Speech Tokenizer (12Hz)        │  → 将音频编码为离散码本 (audio_codes)
│     - 12.5Hz 采样率                  │     - 16层多码本设计
│     - 极端比特率压缩                  │     - 因果卷积网络
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│  2. Speaker Encoder                │  → 提取说话人嵌入 (spk_emb)
│     - x-vector 说话人特征            │     - 16kHz 采样率处理
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│  3. Dual-Track LM 生成              │  → 语音生成
│     - 主轨道: 语义内容               │     - 副轨道: 声学特征
│     - 流式生成 (<97ms 首包延迟)       │     - 支持 ICL / x-vector 两种模式
└─────────────────────────────────────┘
    ↓
生成音频
```

### 两种克隆模式

| 模式 | 输入 | 特点 | 适用场景 |
|------|------|------|----------|
| **ICL (In-Context Learning)** | 参考音频 + 参考文本 + 说话人嵌入 | 保留更多韵律特征、语调、语速 | 高质量克隆 |
| **x-vector Only** | 仅说话人嵌入 | 只保留音色，忽略韵律 | 快速克隆 |

### 关键参数

- **参考音频长度**: 3-10 秒 (推荐)
- **采样率**: 16kHz (模型要求)
- **格式**: WAV 16-bit PCM
- **语言支持**: 10 种语言跨语言克隆 (中文/英文/日文/韩文等)
- **推理设备**: Mac MPS / CUDA / CPU

## 素材质量科学评估

### 原始素材分析 (jarvis_welcome_home.wav)

| 指标 | 数值 | 评估 |
|------|------|------|
| 采样率 | 16,000 Hz | ✅ 符合要求 |
| 总时长 | 12.20 秒 | ⚠️ 过长 |
| **最大振幅** | **0.1482** | ❌ **太低** |
| **静音比例** | **64.6%** | ❌ **过多** |
| RMS 能量 | 0.0165 | ❌ 太低 |
| 有效语音段 | 9 段 | ⚠️ 分散 |
| 最长段时长 | 3.42 秒 | ✅ 合适 |
| 削波采样 | 0 | ✅ 无削波 |

### 问题诊断

1. **音量过低**: 最大振幅仅 0.14 (标准应为 0.9-1.0)
2. **静音过多**: 64.6% 是静音，浪费模型上下文
3. **语音分散**: 9 个短段，模型难以学习完整特征

### 优化处理

```python
# 1. 提取最长非静音段
non_silent = librosa.effects.split(wav, top_db=20)
segment = wav[start:end]  # 3.42s

# 2. 归一化音量
segment = segment / np.max(np.abs(segment)) * 0.95

# 3. 保存优化音频
# 输出: jarvis_clean_3.4s.wav
```

### 优化后素材

| 指标 | 优化后 | 改善 |
|------|--------|------|
| 时长 | 3.42 秒 | ✅ 精简 |
| 最大振幅 | 0.95 | ✅ 标准化 |
| 静音比例 | <1% | ✅ 纯净 |

## 使用建议

### 最佳实践

1. **素材准备**:
   - 提取 3-10 秒纯净语音
   - 去除静音段
   - 音量归一化到 0.9-1.0

2. **克隆命令**:
   ```bash
   ~/.openclaw/tools/qwen-tts-base/qwen-tts-clone \
     "Your text here" \
     jarvis_clean_3.4s.wav \
     output.wav \
     "Welcome home sir"
   ```

3. **跨语言注意**:
   - 英文 → 中文克隆效果不稳定
   - 建议直接使用中文说话人样本
