# Legacy Inline Prompt Template

Updated: 2026-05-14

This file is legacy reference material. New inline images should start from an
Open Visual Grammar canon and a short visual score, then compile into a runtime
prompt under the article package's `_work/images/` folder.

适用于正文插图。

## 结构

```text
主体：{subject}
构图：{composition}
动作/状态：{action_or_tension}
场景：{location}
风格：{style_keywords}
光影：{lighting}
细节质感：{detail_texture}
技术参数：editorial illustration, article inline image, concept-first visual metaphor
不要：{negative_keywords}
```

## 使用原则

- 正文图优先服务章节转折、核心论点、数据关系或抽象概念
- 不要和头图重复同一画面，只保持同一视觉语言
- 如果段落信息密度高，优先做结构化隐喻，而不是堆砌元素
